********Git Commands***************

git init - initialize the file directory into git repo.
git status - show the status of repo. (untracked, staged, tracked)
git add - stagged the untracked files or changes.
git commit - tracked the stagged changes
git rm --cache - delete the files or changes
git restore - restore the deleted changes
git config - set username and email
git switch - used to switch between branches
git checkout -b - used to switch branches as well as create branches
git clone - used to clone git repo
git push - used to push local changes to remote repo
git remote -v - to check the origin addr.
git log - used to checke logs of commit
git pull - pull che changes from remote repo
